# Wikipedia Clone

This project is a simple clone of Wikipedia implemented using HTML, CSS, and Python. It provides a basic version of Wikipedia's functionality, allowing users to search for articles, view article content, and navigate through different categories.

## Overview

The Wikipedia clone consists of HTML files for the front-end interface, CSS files for styling, and Python files for handling backend functionalities such as article search and content retrieval.

### Features

- **Article Search**: Users can search for articles using keywords or phrases.
- **Article Viewing**: Users can view the content of articles in a readable format.
- **Category Navigation**: Users can navigate through different categories to explore articles.

## Files

- **index.html**: Main HTML file containing the structure of the Wikipedia clone's front-end interface.
- **styles.css**: CSS file for styling the HTML elements and providing a visually appealing interface.
- **server.py**: Python file for handling backend functionalities such as article search and content retrieval.

## Styling

The clone features a simple and user-friendly design with minimalistic styling. CSS is used to enhance the readability and visual appeal of the interface.
